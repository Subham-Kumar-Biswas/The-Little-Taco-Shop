# The Little Taco Shop

A multi-page responsive restaurant website.

## Pages
- Home (menu with CSS Grid layout)
- Hours
- Contact (form)
- About

## Features
- Semantic HTML5
- CSS custom properties (variables)
- CSS Grid for menu table
- Dark mode via prefers-color-scheme
- Sticky header and footer
- Keyframe animation on hero section
- Responsive design
- Dynamic copyright year via JavaScript DOM

## Built With
HTML | CSS | JavaScript
